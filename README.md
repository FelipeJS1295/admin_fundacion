# Panel Contabilidad — Fundación

FastAPI + Jinja2 + MySQL. Entradas/Salidas, categorías, usuarios y roles.

## Requisitos
- Python 3.10+
- MySQL
- `pip install -r requirements.txt`

## Configuración
1. Copia `.env.example` a `.env` y completa tus valores.
2. Ejecuta:
   ```bash
   uvicorn app.main:app --reload
